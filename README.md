# 520 情人节惊喜页（微信友好）

浪漫 + 极客风 H5，已针对**微信内置浏览器**优化。

## 微信里怎么打开

1. 把项目部署到 **HTTPS** 静态托管（微信要求外链必须是 https）
   - 例如：GitHub Pages、Vercel、腾讯云 COS、阿里云 OSS 等
2. 获得链接后，发到微信聊天或朋友圈，点击打开即可

本地调试（仅电脑浏览器）：

```bash
cd /Users/gantao/Gitroom/day520
python3 -m http.server 8765
```

手机同一 WiFi 下可访问 `http://电脑IP:8765`（微信内仍建议用 HTTPS 正式地址）。

## 微信适配说明

- 竖屏、安全区（刘海/底部横条）
- 数字键盘输入密码（`type="tel"` + 解锁按钮）
- 去掉 Google 字体，使用系统字体（国内加载更快）
- 矩阵雨背景（微信端自动降帧减负）
- 禁止页面橡皮筋滚动
- 按钮最小点击区域 48px

## 流程

1. 密码默认 **`5201314`**（`app.js` 里 `PASSWORD` 可改）
2. 真实肖像扫描揭示 → 命运转盘
3. 慢停 520 → 弹出「惊喜」→ 快转停在 **1314**

## 分享卡片（可选）

部署后若需微信分享带标题/描述，在 `index.html` 的 `og:title`、`og:description` 已预留；分享缩略图需配置 `og:image` 为**绝对路径**的 HTTPS 图片 URL。
